# Vision Kit 🌿

Este repositorio es un **Vision Kit** compartido para visiones de vida, comunidad y proyecto.

Ofrece un backbone amplio que puede reutilizarse, adaptarse y forkearse — sin obligar a nadie a entrar en un club cerrado ni en un marco rígido.

> **Verdecito lo que salga — “Let it sprout” 🌿**
> 
> Un lente sencillo (opcional), nacido en [Voluntariado Barranco](https://voluntariado.barranco.life): mantener las ideas regenerativas, simples y abiertas a la evidencia.
>

---

- Una [**visión canónica**](./VISION.md)
- Un upstream liviano que puede copiarse, adaptarse o forkearse
- Un backbone vivo para personas, comunidades y proyectos que quieren articular valores, propósito y misión con mayor claridad

## Qué no es este repo

- No es un workflow de proyectos
- No es un registry
- No es un intermediario financiero

## Cómo usarlo

### 1) Haz fork de este repo
Haz fork de este repositorio si quieres mantenerte explícitamente enraizado en este backbone.

### 2) Copia solo lo que necesites
Si no quieres aparecer como fork, copia [`./VISION.md`](./VISION.md) dentro de tu propio repo y adáptalo a tu contexto.
